## Stripe Ecommerce Demo
A PHP and JS application for building a light ecommerce system.
Stripe API Version: 2026-03-25.dahlia

## Structure
- `src/App/` — general application classes
- `src/Ecommerce/` - ecommerce standalone package we're building
- `docs/` — planning, requirements, and reference docs
- `storage/schema.sql` — database schema
- `resources/` - JS modules and CSS
- `views/` - frontend templates and components using Plates

## Code Style
- Use tabs for indentation, not spaces
- PHP and JS files must use tab indentation

## Current Focus
Implementing different checkout flows.
